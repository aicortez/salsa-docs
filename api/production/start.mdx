---
title: "Start Production Batch"
description: "Begin tracking a new salsa production batch"
api: "POST /production/batches"
deprecated: false
version: "2.0"
---

```http
POST /production/batches
```

**Request Body:**

```json
{
  "recipe_id": "rec_456",
  "batch_number": "BATCH-2024-001",
  "planned_quantity": 5.0,
  "production_date": "2024-01-15T08:00:00Z",
  "assigned_chef": "chef_123"
}
```

**Response:**

```json
{
  "id": "batch_789",
  "recipe_id": "rec_456",
  "batch_number": "BATCH-2024-001",
  "status": "in_progress",
  "started_at": "2024-01-15T08:00:00Z",
  "assigned_chef": "chef_123"
}
```
