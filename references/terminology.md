# Product Terminology — Salsa as a Service v2.0

Use this file to ensure correct naming of features and concepts when responding to users.

## Core Platform Terms

**Smart Batch** — A production run initiated through the AI-optimized production flow (Production > Smart Batch). Recommends batch size and timing based on current demand forecasts.

**AI Dashboard** — The main analytics view. Displays real-time metrics enhanced with AI predictions including demand forecasts, quality scores, cost projections, and waste alerts.

**Daily AI Briefing** — An automated overnight summary delivered each morning with the top 3 improvement opportunities, anomalies, and recommended actions.

**AI Report Builder** — Natural language report generation tool. Users describe a report in plain English and AI selects the appropriate metrics and visualizations.

**Forecast Horizon** — The number of days ahead the AI forecasts demand. Options: 7 days (daily planning), 14 days (ordering/staffing), 30+ days (strategic/seasonal).

**Smart Alerts** — AI-configured alerts that learn from patterns rather than static thresholds. Configured under AI Features > Analytics.

## IoT Terms

**IoT Gateway** — A hardware device that connects sensors to the network when direct WiFi/Bluetooth is not available.

**Freshness Score** — A 0–10 metric generated by chemical/RFID sensors indicating ingredient freshness. Minimum acceptable threshold: 8.0.

**Predictive Maintenance** — AI feature that predicts equipment failures 3–14 days in advance with 85% accuracy by analyzing sensor patterns.

**Computer Vision Inspection** — Automated visual QA using cameras. Checks for contamination, inconsistency, portion control, and packaging quality. Covers 100% of production.

## Inventory Terms

**Auto-Reordering** — Automated purchase order generation based on AI forecasts. Configurable approval thresholds (e.g. auto-approve under $1,000, require approval over $1,000).

**Safety Stock Level** — A multiplier (default: 1.5x) applied to reorder calculations to buffer against lead time variance.

**Reorder Trigger** — The logic that initiates a purchase order. Options: `forecast_based` (AI-driven) or `threshold_based` (fixed stock level).

## Quality Terms

**Predictive Quality Score** — An AI-generated quality prediction (0–10 scale) issued before a batch completes. Composite of pH sensor, viscosity, temperature, and visual AI scores.

**Anomaly Detection** — Real-time AI monitoring that flags unusual ingredient combinations, temperature deviations, inconsistent mixing times, or pH variations.

**HACCP Automated Documentation** — Automatic generation and storage of Hazard Analysis Critical Control Points records for compliance.

## API Terms

**v2 Base URL** — `https://api.salsaasaservice.com/v2` (v1.0 deprecated Dec 31, 2025)

**Webhooks v2** — Enhanced webhook system with guaranteed delivery, retry logic, event filtering, payload transformation, and SHA-256 signature verification.

**GraphQL Subscriptions** — Real-time data push via GraphQL (e.g. `OnQualityAlert` subscription for live quality alerts).

## Enterprise Terms

**RBAC** — Role-Based Access Control. v2.0 uses YAML-defined custom roles with granular permissions across locations, features, and data types (PII, financial, operational).

**Location Groups** — Organizational units for managing chains. Allows group-level policies, bulk operations, and regional reporting.

**Recipe Distribution** — Centralized recipe management that pushes updates to selected locations while allowing location-specific modifications.

**Blue-Green Deployment** — The zero-downtime deployment strategy used for v1.0 → v2.0 migrations.
