# Feature Map — Salsa as a Service v2.0

Use this file to identify which area of the product a question relates to, and find the correct docs URL.

## Getting Started
- Account creation, SSO setup, onboarding wizard
- Docs: https://shib.mintlify.app/getting-started
- Supports: email/password + 2FA, Google/Microsoft/Okta SSO, SAML 2.0
- AI setup wizard: analyzes restaurant profile and recommends settings

## AI Demand Forecasting
- Predicts salsa demand 7–30 days in advance
- Achieves 70–80% accuracy with 90 days of data; 95%+ after 6 months
- Integrates: POS, reservations, weather APIs, local event calendars
- Configuration: AI Features > Demand Forecasting
- Docs: https://shib.mintlify.app/advanced-usage#ai-powered-demand-forecasting

## IoT Sensor Integration
- Supported sensors: temperature, RFID/freshness, weight, pH/viscosity/color probes
- Setup: hardware selection → physical install → network config → QR registration → calibration
- Mobile app includes AR visualization for sensor placement
- Docs: https://shib.mintlify.app/getting-started#iot-sensor-integration
- Advanced config: https://shib.mintlify.app/advanced-usage#iot-sensor-integration

## Inventory Management
- AI-powered tracking with predictive stock levels and auto-reordering
- RFID and weight sensors provide real-time levels
- Smart waste prevention: suggests recipes for near-expiry ingredients
- Auto-ordering supports: Sysco, US Foods, Performance Food Group, Gordon Food Service
- Docs: https://shib.mintlify.app/inventory

## Quality Control
- AI predicts quality scores before batch completion (95% accuracy)
- Connected testing: pH meters, viscosity sensors, color sensors, temperature probes
- Computer vision for visual inspection (100% production coverage vs manual sampling in v1.0)
- Critical alerts trigger immediate production halts
- HACCP and FDA compliance automated
- Docs: https://shib.mintlify.app/quality

## Analytics & Reporting
- AI Dashboard: demand forecasts, quality predictions, cost forecasting, waste predictions
- Daily AI Briefing: top 3 opportunities, critical alerts, recommended actions
- Custom reports via natural language (AI Report Builder)
- Export formats: CSV/Excel, PDF, JSON/GraphQL, Power BI/Tableau native connectors
- Docs: https://shib.mintlify.app/analytics

## Integrations
- 50+ POS integrations: Square (OAuth, real-time sync), Toast (v2 API), Clover (App Market)
- Enterprise POS: Oracle Micros, NCR Aloha (custom integration)
- ERP: SAP, Oracle, Microsoft Dynamics
- Supply chain: Sysco, US Foods, PFG, Gordon Food Service
- Docs: https://shib.mintlify.app/integrations

## Enterprise Features
- Multi-location management: centralized dashboard, location groups, recipe distribution
- Advanced RBAC: granular role/permission policies defined in YAML
- Compliance: SOC 2, HACCP, FDA automation
- White-label support
- Inventory transfers between locations
- Docs: https://shib.mintlify.app/advanced-usage#enterprise-features
