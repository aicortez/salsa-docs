## Version 2.5.1 - 16 avril 2026

### 🏗️ Infrastructure

#### Configuration monorepo sous-chemin
- Ajout d'un sous-répertoire `docs/` avec un site de documentation Mintlify dédié pour les guides développeurs internes
- Inclut des guides de démarrage, des références de personnalisation, des guides de rédaction de contenu et des pages d'intégration d'outils IA
- Configuré avec son propre `docs.json`, navigation et endpoints de référence API

### 🐛 Corrections de bugs

- **Spécification OpenAPI** : Correction du préfixe de chemin v2 incohérent sur l'endpoint de suppression — `/plants/{id}` est maintenant correctement `/v2/plants/{id}` pour correspondre à tous les autres endpoints

</Update>

## Version 2.5.0 - 15 mars 2024

### 🚀 Nouvelles fonctionnalités majeures

#### Lancement de l'API GraphQL
- **Endpoint GraphQL** : Nouveau langage de requête flexible pour intégrations avancées
- **Abonnements en temps réel** : Abonnements basés sur WebSocket pour mises à jour de données en direct
- **Introspection de schéma** : API auto-documentée avec explorateur de schéma intégré
- **Optimisation de requêtes** : Optimisation et groupement automatiques de requêtes

#### Prévisions IA améliorées v2
- **Ensemble multi-modèles** : Combine 5 modèles ML pour précision de 98%
- **Prévisions contextuelles** : Incorpore événements locaux, météo, tendances sociales
- **Intervalles de confiance** : Plages de confiance statistiques pour toutes prédictions
- **Analyse de scénarios** : Simulez différents scénarios et leur impact

#### Hub de capteurs IoT v2
- **Edge Computing** : Traitement sur appareil pour temps de réponse plus rapides
- **Réseau maillé** : Les capteurs communiquent directement pour meilleure fiabilité
- **Maintenance prédictive** : L'IA prédit les pannes d'équipement 14 jours à l'avance
- **IA d'inspection visuelle** : Vision par ordinateur pour inspection qualité automatisée

### 🔧 Améliorations majeures

- **Performance** : Temps de réponse API 65% plus rapides avec nouvelle couche de cache
- **Application mobile** : Refonte complète avec architecture hors ligne prioritaire
- **Sécurité** : Ajout du support FIDO2/WebAuthn pour authentification sans mot de passe
- **Intégrations** : Ajout de 15 nouveaux partenaires d'intégration (total 65+)
- **Tableau de bord** : Refonte avec widgets et mises en page personnalisables

### 🐛 Corrections de bugs

- Correction de condition de course rare dans la synchronisation d'inventaire multi-emplacements
- Résolution de fuite de mémoire d'abonnement GraphQL
- Correction de gestion des fuseaux horaires dans calculs de prévision
- Correction du formatage d'export CSV pour formats de nombres internationaux

</Update>

## Version 2.0.0 - 15 juillet 2023

### 🎉 **VERSION MAJEURE : Salsa as a Service v2.0**

La plus grande mise à jour de l'histoire de notre plateforme, v2.0 apporte l'intelligence alimentée par IA, l'intégration IoT et des fonctionnalités de niveau entreprise.

> **Warning:**
**Changements incompatibles** : v2.0 introduit de nouveaux endpoints API et méthodes d'authentification. Voir notre [Guide de migration](https://docs.salsaasaservice.com/migration/v1-to-v2).


#### 🤖 Fonctionnalités alimentées par IA

**Moteur de prévision de demande**
- Modèles d'apprentissage automatique prédisant la demande 7-30 jours à l'avance
- Incorpore données météo, événements locaux, modèles historiques
- Précision de 85-95% après 90 jours d'entraînement
- Planification de production automatisée basée sur prévisions

**IA de prédiction de qualité**
- Prédit les scores de qualité avant l'achèvement du lot
- Détection d'anomalies pour identification précoce des problèmes
- Intégration de tests de qualité automatisés
- Prédiction de durée de conservation

**IA d'optimisation des coûts**
- Identifie opportunités d'économies
- Suggère substitutions d'ingrédients
- Optimise les quantités et timing de commande
- Réduit le gaspillage de 40-60%

#### 📡 Intégration de capteurs IoT

**Surveillance en temps réel**
- Capteurs de température et humidité
- Suivi d'ingrédients basé sur RFID
- Équipement de test de qualité automatisé
- Systèmes de vision pour inspection visuelle

**Maintenance prédictive**
- Prédiction de panne d'équipement 3-14 jours à l'avance
- Planification de maintenance automatisée
- Alertes de dégradation de performance
- Réduction de 75% du temps d'arrêt

#### 🏢 Fonctionnalités d'entreprise

**Gestion multi-emplacements**
- Support illimité d'emplacements
- Tableau de bord centralisé pour tous emplacements
- Configurations spécifiques par emplacement
- Transferts d'inventaire inter-emplacements

**Sécurité avancée**
- Single Sign-On (SSO) avec SAML 2.0
- Authentification multi-facteurs (MFA)
- Support d'authentification biométrique
- Contrôle d'accès basé sur rôles (RBAC)
- Certification SOC 2 Type II

#### 🔗 Intégrations étendues

**50+ intégrations pré-construites**
- Systèmes POS majeurs (Square, Toast, Clover, etc.)
- Distributeurs alimentaires (Sysco, US Foods, etc.)
- Systèmes ERP (SAP, Oracle, Microsoft Dynamics)
- Logiciels de comptabilité (QuickBooks, Xero, Sage)
- Plateformes de communication (Slack, Teams)

**Plateforme développeur**
- API RESTful avec documentation complète
- Système de webhooks pour événements en temps réel
- SDKs officiels pour 8+ langages
- Environnement sandbox pour tests

### 📚 Migration depuis v1.0

**Migration sans temps d'arrêt**
- Outil de migration automatisé
- Validation et vérification des données
- Période d'exécution parallèle (v1 et v2)
- Équipe de support de migration dédiée
- Sessions de formation gratuites

**Calendrier de migration**
- Migration typique : 1-2 semaines
- Migrations entreprise : 2-4 semaines
- Support v1.0 jusqu'au 31 décembre 2025

> **Note:**
**Offre de lancement v2.0** : Les clients v1.0 existants reçoivent 3 mois gratuits lors de la mise à niveau vers les plans Pro ou Entreprise v2.0.
