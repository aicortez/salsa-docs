# Getting Started with Salsa as a Service v2.0

Welcome to v2.0! This guide will help you set up your account and leverage the new AI-powered features, IoT sensors, and advanced integrations.

## Prerequisites

Before you begin, ensure you have:

- A valid business email address
- Access to your restaurant's systems (POS, inventory, etc.)
- Basic knowledge of your current salsa operations
- Admin access to configure integrations
- <Badge color="green">New in v2.0</Badge> Optional IoT sensor hardware for quality monitoring

## AI Feature Configuration

### Demand Forecasting Setup

Enable AI-powered demand forecasting to predict salsa needs 7-30 days in advance:

<Steps>
  <Step title="Enable Forecasting">
    Go to **AI Features \> Demand Forecasting** and toggle the feature on.
  </Step>
  <Step title="Connect Data Sources">
    Link your POS system, reservation system, and any external data sources (weather APIs, local event calendars).
  </Step>
  <Step title="Historical Data Import">
    Import at least 90 days of historical sales data for optimal accuracy. The AI improves over time.
  </Step>
  <Step title="Set Forecast Parameters">
    Configure forecast horizon (7, 14, or 30 days), confidence intervals, and alert thresholds.
  </Step>
  <Step title="Review Initial Forecast">
    The AI generates your first forecast within 24 hours. Review and adjust as needed.
  </Step>
</Steps>

<Note>**Forecast Accuracy**: The AI achieves 70-80% accuracy with 90 days of data and improves to 95%+ accuracy after 6 months of learning.</Note>

### ERP System Integration

Connect v2.0 with your enterprise resource planning system:

<CodeGroup dropdown>

```javascript
// Example: ERP integration configuration
import { SalsaClient } from 'salsa_sdk';

const client = new SalsaClient({ apiKey: 'your_api_key' });

async function connectErp() {
  const erpConnection = await client.integrations.connectErp({
    system: 'sap',
    credentials: {
      host: 'erp.yourcompany.com',
      client: '100',
      username: 'SALSA_INT',
      auth_type: 'oauth'
    },
    sync_options: {
      inventory: true,
      procurement: true,
      cost_accounting: true,
      frequency: 'real-time'
    }
  });

  console.log(`ERP connected: ${erpConnection.status}`);
}

connectErp().catch(console.error);
```

```python
# Example: ERP integration configuration
from salsa_sdk import SalsaClient

client = SalsaClient(api_key="your_api_key")

# Connect to SAP
erp_connection = client.integrations.connect_erp(
    system="sap",
    credentials={
        "host": "erp.yourcompany.com",
        "client": "100",
        "username": "SALSA_INT",
        "auth_type": "oauth"
    },
    sync_options={
        "inventory": True,
        "procurement": True,
        "cost_accounting": True,
        "frequency": "real-time"
    }
)

print(f"ERP connected: {erp_connection.status}")
```

```php
// Example: ERP integration configuration (PHP)
require 'vendor/autoload.php';

use SalsaSdk\SalsaClient;

$client = new SalsaClient(['api_key' => 'your_api_key']);

$erpConnection = $client->integrations->connectErp([
    'system' => 'sap',
    'credentials' => [
        'host' => 'erp.yourcompany.com',
        'client' => '100',
        'username' => 'SALSA_INT',
        'auth_type' => 'oauth'
    ],
    'sync_options' => [
        'inventory' => true,
        'procurement' => true,
        'cost_accounting' => true,
        'frequency' => 'real-time'
    ]
]);

echo "ERP connected: " . $erpConnection->status . PHP_EOL;
```
